Objetivo

Simular os cinco níveis de um reservatório de água.

Exibir mensagens de alerta com cores distintas para cada nível.

Demonstrar o uso da biblioteca colorama para destacar informações no terminal.

Funcionalidades

Lista com cinco níveis de alerta: Muito baixo, Baixo, Médio, Alto e Muito alto.

Função que associa cada nível a uma cor específica.

Impressão no terminal das mensagens com cores correspondentes.

Restauração automática do estilo padrão após cada mensagem.
